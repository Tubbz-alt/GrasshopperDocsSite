Mega Write

Component Index > Firefly > Arduino & I/O Boards > Mega Write

This component will write values to all corresponding digital
pins on an Arduino Mega board. Note: This component is
meant to be used in tandem with the Firefly Firmata Arduino
Sketch which can be downloaded from the Firefly website
at: http://www.fireflyexperiments.com.

Colour HSV

Component Index Grasshopper DisplayColourHSV

Create a colour from floating point {HSV} channels.

Inputs

NameIDDescriptionType
AlphaAAlpha channel (alpha is defined in the range {0.0 to 1.0})Number
HueHHue channel (hue is defined in the range {0.0 to 1.0})Number
SaturationSSaturation channel (saturation is defined in the range {0.0 to 1.0})Number
ValueVValue channel (value/brightness is defined in the range {0.0 to 1.0})Number

Outputs

NameIDDescriptionType
ColourCResulting colourColour

Bouncers

Component Index > Nudibranch > Animators > Bouncers

Generates and animates particles moving within a user defined box (Timer defined).

Inputs

NameIDDescriptionType
BoundaryBBox BoundaryBox
NumberBNNumber of BouncersInteger
FactorFBouncer's velocity factorNumber
SeedSRandom seed numberInteger
WrapWToggle boundariesBoolean
RandomizeRaRandomize the speed for each particle(Domain: -Factor to Factor)Boolean
RunRToggle to run/stop the animationBoolean

Outputs

NameIDDescriptionType
ParticlesPThe generated point list per iterationPoint
VectorsVThe generated vector list per iterationVector

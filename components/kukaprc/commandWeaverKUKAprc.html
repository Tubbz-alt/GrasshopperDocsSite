Command Weaver - KUKA|prc

Component Index KUKA Prc05 | UtilitiesCMD WEAVE

This component will weave together KUKA|prc commands. You may require a certain sequence of events, e.g. move PTP, open gripper, move LIN down, close gripper, move LIN up... When using the Command Weaver, put in all move PTP commands in the first input, all open gripper commands in the second input etc. The command weaver will then take the first item from the first input, then the first item from the second input... weaving together the command structure. If you enable the tree option, weaving will take place according to tree branches, instead of individual items.

UDP Sender

Component Index gHowlUDP

Allows to send data through the network to any computer

Inputs

NameIDDescriptionType
Address@The IP address to send to. If empty, the loopback address is usedString
PortPThe port number to sent to, on the IP aboveInteger
Pattern#The pattern:\r\n 0 = sends data as ASCII text (7),\r\n 10 = sends data as array of doubles (8)\r\n999 = Sends OSC Message\r\nInteger
DataDThe string to sendGeneric

Outputs

NameIDDescriptionType
InfoINetwork informationString

Parameter Subdivide Surface

Component Index PufferfishSurfaceParamSubDivSrf

Generate a grid of subdivision surfaces on a surface from domain dividing {u} and {v} parameters.

Inputs

NameIDDescriptionType
SurfaceSSurface to divideSurface
Parameters UuDivision parameters along surface {u} domain (0.0 = {u}start, 1.0 = {u}end) For best results {u} parameters should be between 0 and 1, in numerical order, and non duplicateNumber
Parameters VvDivision parameters along surface {v} domain (0.0 = {v}start, 1.0 = {v}end) For best results {v} parameters should be between 0 and 1, in numerical order, and non duplicateNumber

Outputs

NameIDDescriptionType
SubdividedSResulting subdivision surfacesSurface

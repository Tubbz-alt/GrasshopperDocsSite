Diameter Dimension

Component Index > FabTools > Annotation > Diameter Dimension

This component creates a diameter dimension from circles and arcs.
The resulting RadialDimension object can be baked with the Advanced Bake (by blickfeld7.com)
Version 0.92

Inputs

NameIDDescriptionType
Circle / ArcAThe circle or arc geometry to create diameter dimension annotation.Arc
PositionPThe position for the dimensional arrow.Point
View Direction (Optional)VSet a Vector to force the direction of the dimensional text to a specific direction. By default the world x-axis is used to display the text.Vector

Outputs

NameIDDescriptionType
Diameter DimensiondimThe RadialDimension object.Goo

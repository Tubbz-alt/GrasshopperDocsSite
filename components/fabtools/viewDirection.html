View Direction

Component Index FabToolsUtilsViewDir

Gets the vector of the view direction from the active viewport or a specified view. (by blickfeld7.com)
Version 0.92

Inputs

NameIDDescriptionType
RefreshRRefreshes the view direction vector.Boolean
(Optional) ViewportVBy default the active viewport is choosen. Provide the name of the viewport from a Panel, or connect a ValueList to get a list of the current available Views.String

Outputs

NameIDDescriptionType
VectorVThe view direction vector of the active/specified viewport.Vector
